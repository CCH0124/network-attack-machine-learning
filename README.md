## 目錄
- data-preprocessing 
    - 資料前處理
- feature_selection
    - 特徵選取
- figure
    - 存放神經網路訓練的指標圖片
- model_val
    - 用來存放關於訓練模型的資訊，用來觀察模型好壞
- pre_train
    - 訓練模型的檔案
- trainLog
    - 訓練的日誌檔
- utils
    - 存放一些模型架構或是資料切割等方法
